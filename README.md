# MysteryBox – Nettbutikk for tilpassede overraskelsesbokser

Dette prosjektet er en nettside som lar brukeren lage sin egen Mystery Box.
Brukeren kan velge: mottaker, kjønn, alder, størrelse, tema og personlig melding.

Prosjektet er bygget med:
- HTML
- CSS
- JavaScript

Prosjektet skal kjøres på Raspberry Pi via GitHub-deploy.

Se dokumentasjon i mappen docs/.
